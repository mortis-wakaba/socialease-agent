# Dorm Conflict Communication Guide

宿舍沟通适合使用具体、可观察的描述，避免直接给对方贴标签。表达顺序可以是：事实、影响、请求。

示例结构：

- 事实：“最近晚上十二点后还有比较大的声音。”
- 影响：“我第二天早课会比较难集中。”
- 请求：“能不能在十二点后把声音放低一些？”

如果担心冲突升级，可以先选择低强度方式，例如文字草稿、找中立同学演练，或在双方都比较平静的时间开口。练习目标是表达边界，而不是控制对方反应。

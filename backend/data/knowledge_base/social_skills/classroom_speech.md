# Classroom Speech Practice Guide

课堂发言练习可以从一个非常小的表达开始。先写下一句核心观点，再补充一个简短理由。目标不是表现完美，而是在安全范围内完成一次清楚表达。

可用步骤：

- 准备一句 20 字以内的开场。
- 先在纸上写，再低声练习。
- 发言时先说核心观点，再说一个理由。
- 结束后记录焦虑强度变化，而不是只评价表现好坏。

如果发言时卡住，可以使用过渡句：“我先说一个初步想法，可能还不完整。”这类句子能降低表达压力，也不会承诺必须一次说完美。

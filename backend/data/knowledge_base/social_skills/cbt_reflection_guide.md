# CBT Style Reflection Guide

CBT 风格反思在本项目中只是自助练习，不是治疗。它用于把社交压力拆成几个栏目：情境、自动想法、情绪、强度、支持证据、反对证据、替代想法和下一步行动。

填写时可以保持简短：

- 情境：发生了什么具体场景。
- 自动想法：脑中最强烈的一句话。
- 情绪和强度：例如焦虑 7/10。
- 证据：分别写支持和反对这个想法的事实。
- 替代想法：更平衡但不强行积极的一句话。
- 下一步：一个小而安全的行动。

如果信息不足，可以先补充情绪强度和下一步行动，不需要一次写得完整。
